# Changelog

Todos los cambios notables de este proyecto se documentarán en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es/1.0.0/),
y este proyecto adhiere a [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2024-01-XX

### Añadido

- Arquitectura refactorizada con TypeScript
- Estado global con Zustand
- Sistema de exportación/importación de datos
- Nuevas plantillas predefinidas
- Atajos de teclado completos
- Soporte PWA mejorado
- Documentación profesional (README, CONTRIBUTING, LICENSE)
- Componentes modulares y reutilizables
- Sistema de tipos completo

### Cambiado

- Migración completa a TypeScript
- Mejoras significativas en UI/UX
- Paleta de colores unificada
- Rendimiento optimizado
- Estructura de proyecto profesional

### Corregido

- Persistencia de datos más robusta
- Manejo de errores mejorado
- Accesibilidad mejorada

## [1.0.0] - 2024-01-XX

### Añadido

- Gestión de plantillas de prompts
- Editor de contexto con variables
- Backlog integrado
- Sistema de favoritos
- Categorías de plantillas
- Búsqueda y filtrado
- Persistencia local
- Soporte PWA básico

---

## Tipos de cambios

- `Añadido` para nuevas características
- `Cambiado` para cambios en características existentes
- `Deprecado` para características que se eliminarán
- `Eliminado` para características eliminadas
- `Corregido` para correcciones de bugs
- `Seguridad` para vulnerabilidades corregidas
